# Url Shortener

## To get started:
  1. Clone this repo locally;
  2. `cd url-shortener`
  3. `npm install` to install its dependencies
  4. `npm run start`

## Installing Mongo using Homebrew

To install Mongo:<br>
`brew tap mongodb/brew`<br>
`brew install mongodb-community`<br>
`brew services start mongodb-community`<br>
